# abtnode

### Project introduce

本项目使用到的技术栈有：

- vuejs 2.x
- vue-router
- @vue/composition-api
- vuetifyjs 2.x
- @mdi/font
- axios
- dayjs

单元测试框架采用 jest

### Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies for production

```
npm run build
```

### Run your unit tests

```
npm run test:unit
```
